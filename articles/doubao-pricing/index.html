国产 AI 的
价格屠夫时刻

豆包 68 元/月对标 ChatGPT Plus、阿里云首创算力分时定价夜间 2 折、Cursor 发布 Agent 专用 Git——AI 正在从"奢侈品"变成"公用事业"。

No.002 2026.06.25 约 5 分钟阅读

6 月 24 日，字节跳动给国产 AI 订阅市场投下了一颗定价炸弹。

豆包全线升级至 Doubao-Seed 2.1 大模型，日均 Token 调用量突破 180 万亿，支持数百个 AI 智能体同步协作，还接入了打车等生活服务功能。同步推出三档付费订阅：标准版 68 元/月、加强版 200 元/月、高级版 500 元/月。

对比一下：ChatGPT Plus 是 135 元/月，Pro 是 677 元/月。豆包的价格几乎是 ChatGPT 的一半甚至三分之一，而且免费用户的权益完整保留——不砍功能、不限制模型、不搞逼氪。

阿里云的"夜间 2 折"

同一天，阿里云给出了更有意思的答案。

桌面智能工具 QoderWork 上线了国内首个 AI 算力分时定价机制：每晚 22 点至次日 8 点，调用 Qwen3.7-Max 旗舰模型低至 2 折。重度用户的批量长任务成本可以削减 60%-80%，官方推荐的使用方式是："白天编辑指令，夜间 AI 自主运行"。

这是一个极具想象力的定价创新。它意味着 AI 算力正在从"统一计价"走向"峰谷分时"，像电力一样成为公用事业。电网有波峰波谷，算力也有——白天大家都在用，贵一点；晚上机器闲着也是闲着，便宜点卖。

"AI 算力的终极形态，就是电力。你不会问'这个月用了多少瓦特'，你只会问'这个月电费多少钱'。" —— 某云厂商产品负责人

Agent 时代的基础设施

价格下降的同时，基础设施也在为 Agent 时代重构。

Cursor 在 Compile 开发者大会上发布了 Origin——专为 AI 智能体设计的云原生 Git 平台。传统 GitHub 是为人类程序员设计的：Pull Request、Code Review、合并冲突解决，都是基于"人会看代码"的假设。但当成百上千个 Agent 同时读写代码库时，传统 Git 根本扛不住。Origin 解决的就是这个问题：自动处理合并冲突、自动修复 CI 故障、自动回滚有问题的提交。

AI 视频领域同样在发生范式转移。HeyGen 开源了 Hyperframes 框架（用 HTML 写视频），OpenMontage 发布了全球首个开源代理式视频制作系统——12 条流水线、52 个集成工具、500+ 代理技能。视频制作从手动剪辑转向代码驱动自动化，一个人加一群 Agent 就能产出过去需要十来人团队才能完成的视频内容。

国产 AI 的商业化路径

豆包和阿里云的定价策略，折射出国产 AI 的独特商业化路径。

美国 AI 公司走的是"高端收割"路线：先服务愿意付高价的企业客户和专业用户，再逐步下沉。但中国市场的逻辑完全不同——这里有全球最庞大的互联网用户群体、最激烈的价格竞争、最成熟的移动支付生态，"低门槛拉新+分层变现"才是被反复验证的王道。

68 元/月是什么概念？一杯奶茶加一份外卖的钱。这个定价不是在和 ChatGPT 竞争，它是在和视频会员、音乐会员、网盘会员竞争——把 AI 从"专业工具"变成"国民应用"。

更关键的是信号意义：国产 C 端 AI 正在走出"免费烧钱换用户"的阶段，开始认真思考怎么赚钱。当价格打到 ChatGPT 的三分之一仍然有利润空间，说明国产模型的推理成本优化已经做到了极致。这种成本优势，最终会转化为全球市场的竞争力。

AI 的公用事业化

68 元/月、夜间 2 折、Agent 专用 Git——这些看似不相关的新闻，指向同一个趋势：AI 正在从"奢侈品"变成"公用事业"。

就像电力从工厂专属变成家家户户都用得起，就像宽带从高端商务变成民生基础设施，AI 算力也在走同样的路。当调用一次大模型的成本比发一条短信还便宜，当 Agent 可以 7×24 小时以极低的成本为你工作，真正的 AI 普及才会到来。

价格屠夫不是坏事。它把技术从象牙塔拉到寻常百姓家，让更多人用得起、用得上。豆包 68 元的背后，是千万级用户的市场，是 Agent 大规模商用的基础，是中国 AI 生态独特的竞争力。

白天写指令，夜里让 Agent 干活，账单上的电费比咖啡还便宜——这一天可能比我们想象的更近。
