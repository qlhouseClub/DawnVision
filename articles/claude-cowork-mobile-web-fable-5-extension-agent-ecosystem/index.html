以前要用Claude Cowork，你必须坐在电脑前打开VS Code，通过Claude Code插件才能访问。从7月8日开始，你在地铁上、在会议室里、在午餐排队时，掏出手机就能和你的Claude Cowork继续工作。

Anthropic正式宣布Claude Cowork登陆移动和网页端，首先向Max订阅用户开放，其他付费计划用户将在"未来几周"逐步获得访问权限。同一天Anthropic还宣布，火爆的Fable 5创造性协作体验将在所有付费计划上延长访问至7月12日。两个动作放在一起看，Anthropic的Agent生态闭环正在加速成型。

从IDE工具到全场景协作平台

理解Claude Cowork跨平台的意义，要先理解Cowork在Anthropic产品矩阵中的定位。

Claude Code是给开发者写代码用的IDE Agent；Claude.ai的普通对话是通用问答；Claude Cowork则是Anthropic面向团队协作场景推出的企业级Agent平台——你可以给Cowork配置团队知识库、连接工作工具、设定工作流程，让AI Agent真正作为团队的一员参与日常工作。之前Cowork最大的局限是只能通过Claude Code在IDE里访问，这意味着它主要被开发者使用，产品经理、运营、市场、销售这些非技术角色根本接触不到。

移动和网页端的开放彻底改变了这个局面。跨平台意味着Cowork从一个开发者工具变成了一个全团队、全场景可用的协作平台。产品经理在客户现场可以用手机让Cowork整理会议纪要，运营在活动现场可以让Cowork实时生成社交媒体文案，管理者在出差路上可以让Cowork汇总本周各项目进度——这些场景以前都需要回到电脑前才能完成，现在掏出手机就能做。

Fable 5延长+开源不构成威胁，Anthropic的护城河逻辑

Fable 5的延长访问同样值得关注。Fable是Anthropic推出的创造性多Agent协作体验——多个AI角色在共享工作空间里协同创作，从写剧本到做设计到策划方案，用户作为导演引导多个Agent一起工作。Fable 5上线后在开发者和创意人群体中爆火，Hacker News帖子获得170分和173条评论，很多人称之为"第一次感受到AI真正有了协作感"。

原定于7月8日结束的访问被延长到7月12日，说明Anthropic对Fable的用户粘性和转化效果很满意，正在考虑更长期的产品化方案。

同一天TechCrunch发表了一篇有意思的分析：为什么开源AI的崛起目前还没有伤害到Anthropic。文章的核心论点是开源模型和前沿模型其实在捕获同一个生命周期的两个不同阶段——开源模型在推理成本和本地化部署上有优势，但前沿模型在最复杂的推理任务、Agent能力、多模态理解上依然保持代际差距。企业客户愿意为Claude Opus和Sonnet的能力付费，不是因为他们不知道有便宜的开源模型，而是因为最有价值的工作任务需要最强的模型能力。

"开源模型赢了成本敏感型场景，闭源前沿模型赢了价值敏感型场景——这两个市场目前不是零和博弈。"—— TechCrunch分析文章

Claude Cowork的跨平台和Fable的火爆印证了这个判断。Anthropic的护城河不只是模型本身，而是围绕模型构建的完整体验——Cowork的协作工作流、Fable的多Agent创作体验、Claude Code的深度IDE集成、MCP生态的工具连接能力。这些体验层的东西是开源模型短期内复制不了的。

从Claude Code到Claude Cowork移动版，Anthropic正在一步步把Claude从一个聊天机器人变成一个真正的工作操作系统。当你的AI同事可以在任何设备上、任何场景下和你协作时，"AI Agent"这个词就不再是一个概念，而是你每天工作的一部分。

明天见。

Previously, to use Claude Cowork you had to sit at your computer, open VS Code, and access it through the Claude Code plugin. Starting July 8, you can pull out your phone and continue working with your Claude Cowork on the subway, in a meeting room, or while waiting in line for lunch.

Anthropic officially announced that Claude Cowork is launching on mobile and web, rolling out first to Max subscribers with access gradually expanding to other paid plans "in the coming weeks." The same day, Anthropic also announced that the wildly popular Fable 5 creative collaborative experience would be extended across all paid plans through July 12. Taken together, these two moves show Anthropic's Agent ecosystem closing its loop at an accelerating pace.

From IDE Tool to Full-Scenario Collaboration Platform

To understand the significance of Cowork going cross-platform, you first need to understand Cowork's position in Anthropic's product matrix.

Claude Code is an IDE Agent for developers writing code; regular Claude.ai chat is general-purpose Q&A; Claude Cowork is Anthropic's enterprise-grade Agent platform for team collaboration scenarios — you can configure Cowork with team knowledge bases, connect work tools, set workflows, and have the AI Agent truly participate in daily work as a member of the team. Previously, Cowork's biggest limitation was being accessible only through Claude Code in the IDE, meaning it was used primarily by developers, while product managers, operations, marketing, sales, and other non-technical roles had essentially no access.

Opening mobile and web access changes this completely. Cross-platform means Cowork goes from a developer tool to a full-team, full-scenario collaboration platform. PMs can use their phones at client sites to have Cowork organize meeting notes; operations folks can have Cowork generate real-time social media copy at events; managers can have Cowork summarize weekly project progress on business trips — scenarios that previously required being back at a computer can now be done by pulling out a phone.

Fable 5 Extension + Why Open Source Isn't Hurting Anthropic Yet

Fable 5's extended access is equally notable. Fable is Anthropic's creative multi-agent collaborative experience — multiple AI characters collaborate in a shared workspace on everything from screenplays to design to campaign planning, with the user as director guiding multiple Agents working together. Fable 5 exploded in popularity among developers and creative communities after launch; the Hacker News post hit 170 points with 173 comments, with many calling it "the first time AI truly felt collaborative."

Access originally scheduled to end July 8 has been extended to July 12, indicating Anthropic is pleased with Fable's user stickiness and conversion effects and is considering longer-term productization plans.

The same day, TechCrunch published an interesting analysis: why the rise of open source AI isn't hurting Anthropic yet. The core argument: open source models and frontier models are actually capturing two different phases of the same lifecycle — open source models have advantages in inference cost and local deployment, but frontier models maintain a generational gap on the most complex reasoning tasks, Agent capabilities, and multimodal understanding. Enterprise customers are willing to pay for Claude Opus and Sonnet's capabilities not because they don't know about cheap open source alternatives, but because the highest-value work tasks require the strongest model capabilities.

"Open source wins cost-sensitive scenarios; closed frontier models win value-sensitive scenarios — these two markets aren't zero-sum right now."— TechCrunch analysis

Claude Cowork going cross-platform and Fable's popularity confirm this assessment. Anthropic's moat isn't just the model itself; it's the complete experience built around the model — Cowork's collaboration workflows, Fable's multi-agent creative experience, Claude Code's deep IDE integration, and the MCP ecosystem's tool connectivity. These experience-layer elements are things open source models can't replicate in the short term.

From Claude Code to Claude Cowork mobile, Anthropic is steadily transforming Claude from a chatbot into a true work operating system. When your AI colleague can collaborate with you on any device, in any scenario, the term "AI Agent" stops being a concept and becomes part of how you work every day.
