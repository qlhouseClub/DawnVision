# Dawn Vision

> 在噪音中看见信号 — AI时代的深度观察

Dawn Vision 是一个每日更新的 AI 资讯深度阅读站点。每个工作日，我们从海量 AI 资讯中去噪、筛选、深度加工，输出可以直接读完的完整文章——不做信息搬运工，做噪音中的信号过滤器。

## 特点

- **完整文章**：不是摘要+外链，每篇都是独立完整的深度文章
- **杂志风排版**：精心设计的编辑级排版，Source Serif 4 + Noto Serif SC，移动端适配
- **双栏目**：深度观察（正经分析）+ 槽/CAO（AI圈趣事吐槽）
- **零依赖**：纯静态 HTML + CSS，运行在 GitHub Pages 上
- **暗色模式**：自动跟随系统偏好

## 栏目

- **封面故事**：每期一篇长文深度分析（~1500字）
- **深度观察**：短文快速解读（~600字）
- **槽/CAO**：幽默吐槽、AI圈趣事

## 发布节奏

- 周一至周五每日更新
- 采集：09:30 / 12:00 / 14:30
- 深度加工与发布：16:30

## 技术栈

- 纯 HTML + CSS（零JavaScript框架，零构建步骤）
- GitHub Pages 托管
- Google Fonts（Source Serif 4 / Noto Serif SC / PT Serif）
