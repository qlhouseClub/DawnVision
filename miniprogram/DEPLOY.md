# 小程序测试与提交审核指南

## 一、准备工作

### 1.1 注册微信小程序账号

1. 访问 [微信公众平台](https://mp.weixin.qq.com/)
2. 点击「立即注册」，选择「小程序」
3. 按照指引完成账号注册（需要一个未绑定过公众号/小程序的邮箱）
4. 完成主体认证：
   - 个人主体：身份证 + 微信实名认证（免费，适合初期）
   - 企业主体：营业执照（需要300元认证费，可开通支付功能）

**建议**：初期使用个人主体即可，后续可升级为企业主体。

### 1.2 获取 AppID

1. 登录微信公众平台
2. 进入「开发」->「开发管理」->「开发设置」
3. 复制 AppID（类似 `wx1234567890abcdef`）
4. 替换 `project.config.json` 中的 `appid` 字段

### 1.3 下载微信开发者工具

下载地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

选择对应操作系统的稳定版安装即可。

## 二、本地开发测试

### 2.1 安装依赖

打开命令行（PowerShell 或 CMD），进入项目目录：

```bash
cd d:\WorkSpace\DawnVision\miniprogram
npm install
```

如果 npm install 速度慢，可以使用国内镜像：

```bash
npm install --registry=https://registry.npmmirror.com
```

### 2.2 启动开发模式

```bash
npm run dev:weapp
```

等待构建完成（首次构建可能需要1-2分钟）。

### 2.3 在微信开发者工具中预览

1. 打开微信开发者工具
2. 选择「导入项目」
3. 项目目录选择：`d:\WorkSpace\DawnVision\miniprogram`
4. AppID 填入你自己的 AppID（或选择测试号）
5. 点击「导入」
6. 项目加载后，即可在模拟器中看到小程序效果

### 2.4 真机调试

1. 在微信开发者工具中点击「预览」
2. 使用微信扫描二维码
3. 即可在手机上真实体验小程序

## 三、小程序备案（重要）

根据微信要求，2023年9月起所有小程序必须完成备案才能上线。

### 3.1 备案条件

- 已注册小程序账号
- 有已备案的域名（你的 dawnvision.cn 已备案，符合要求）

### 3.2 备案流程

1. 登录微信公众平台
2. 进入「设置」->「基本设置」->「小程序备案」
3. 点击「去备案」
4. 按照指引填写信息：
   - 主体信息（个人/企业）
   - 负责人信息
   - 小程序信息
   - 上传相关证件材料
5. 等待微信审核（通常1-7个工作日）

### 3.3 注意事项

- 备案期间小程序可以开发和测试，但不能正式发布
- 个人主体备案需要人脸核验
- 确保 dawnvision.cn 的备案主体与小程序主体一致

## 四、提交审核前检查清单

### 4.1 基础信息

- [ ] 小程序名称：DawnVision 或 Dawn Vision
- [ ] 小程序头像：使用克莱因蓝背景+白色DV标志（需要设计）
- [ ] 小程序介绍：AI深度观察日刊，穿越嘈杂，洞见留声
- [ ] 小程序类目：选择「资讯」->「资讯信息」或「教育」->「教育信息」
- [ ] 服务标签：科技、AI、资讯

### 4.2 功能检查

- [ ] 首页正常加载，显示最新文章
- [ ] 文章列表可以正常浏览
- [ ] 文章详情页富文本渲染正常
- [ ] 吐槽页显示正常
- [ ] 关于页内容完整
- [ ] TabBar 切换正常
- [ ] 页面跳转正常
- [ ] 分享功能正常

### 4.3 内容合规检查

微信对内容审核非常严格，务必确保：

- [ ] 所有文章内容符合法律法规
- [ ] 没有敏感政治内容
- [ ] 没有色情、暴力内容
- [ ] 没有虚假信息
- [ ] 吐槽栏目内容不过于尖锐，保持理性批评
- [ ] 引用来源标注清晰

### 4.4 隐私协议

根据微信要求，需要提供：

1. **用户隐私保护指引**：
   - 在微信公众平台「设置」->「用户隐私保护指引」中填写
   - 说明收集哪些用户数据（当前版本不收集用户数据，仅本地阅读）
   - 说明数据用途

2. 如果后续接入CloudBase，需要补充：
   - OpenID/UnionID 用于用户识别
   - 收藏/喜欢数据存储

当前版本不收集任何用户数据，可以声明：
> 本小程序仅提供内容阅读服务，不收集、不存储任何用户个人信息。

## 五、提交审核

### 5.1 构建生产版本

```bash
npm run build:weapp
```

构建产物在 `dist/` 目录。

### 5.2 上传代码

1. 在微信开发者工具中，确认代码是最新构建的
2. 点击右上角「上传」
3. 填写版本号（如 1.0.0）
4. 填写项目备注（如 首次发布）
5. 点击「上传」

### 5.3 提交审核

1. 登录微信公众平台
2. 进入「版本管理」
3. 在「开发版本」中找到刚上传的版本
4. 点击「提交审核」
5. 填写审核信息：
   - 功能页面：列出所有tab页面
   - 功能描述：AI行业资讯阅读小程序
   - 测试账号：如果有登录功能需要提供（本小程序不需要）
6. 提交审核

### 5.4 审核时间

- 通常 1-7 个工作日
- 首次审核可能较慢
- 如果被拒，根据拒绝原因修改后重新提交

---

## 六、常见审核拒绝原因及解决方案

### 6.1 「类目选择不正确」

- 解决方案：确保选择了正确的资讯类目，补充类目资质（如果需要）

### 6.2 「内容涉及敏感信息」

- 解决方案：检查文章内容，移除或修改敏感内容
- 吐槽栏目尤其需要注意措辞

### 6.3 「缺少隐私协议」

- 解决方案：在「设置」->「用户隐私保护指引」中补充完整

### 6.4 「功能不完善」

- 解决方案：确保所有Tab页面都有真实内容，不要有空白页或占位页

### 6.5 「需要增值电信业务经营许可证」

- 如果只是展示自己生产的内容，通常不需要
- 如果有UGC（用户生成内容），可能需要办理ICP许可证
- 初期建议关闭评论、用户发布等功能，规避此问题

## 七、发布上线

审核通过后：

1. 在微信公众平台「版本管理」中
2. 审核版本会显示「审核通过」
3. 点击「发布」
4. 选择发布方式：
   - 全量发布：所有用户立即更新
   - 灰度发布：逐步推送给用户（推荐首次发布使用）

发布后约10-30分钟，用户即可搜索到你的小程序。
